# motif

**Personal skill toolkit for semaclaw** — a collection of reusable agent skills for frequently used workflows and tasks.

Each skill is a `.md` file that gives an agent a specialized capability: a repeatable procedure, domain knowledge, or tool integration it can invoke on demand.

## Status

Skills are currently under development. More coming soon.

## Structure

```
motif/
  skills/          # skill definitions (.md files)
  .semaclaw-plugin/
    plugin.json    # plugin metadata
```
