# timbre

**Subagent persona collection for semaclaw** — a curated set of virtual worker personalities you can assign to subagents for specialized tasks.

Each persona is a `.md` file that defines a character's voice, values, and working style. When loaded as a subagent, it shapes how the agent reasons, communicates, and approaches problems.

## Usage

Assign a persona file as the system prompt for a subagent in your semaclaw workflow. The persona will guide the agent's tone, priorities, and problem-solving approach throughout the session.
